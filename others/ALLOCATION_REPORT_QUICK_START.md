## Troubleshooting

### "Export failed" error
1. Check your internet connection (API needs to reach backend)
2. Check browser console (F12) for error message
3. Ensure API server is running: `python3 -m uvicorn api.main:app --port 8000`

### PDF doesn't open
1. Verify you have a PDF reader installed (Adobe Reader, Preview, etc.)
2. Try opening with a different PDF app
3. Check if file was downloaded to your Downloads folder

### DOCX doesn't open
1. Verify Word or Google Docs is installed
2. Try uploading to Google Drive and opening in Google Docs
3. Check if file was downloaded to your Downloads folder
4. If still fails, the server may not have Node.js installed — contact admin

### Export button is greyed out
1. Make sure you've completed the allocation (reached final step)
2. Ensure allocation data is loaded correctly
3. Refresh the page and try again

## Data Privacy

The allocation report contains:
- ✅ Chapter names and allocation numbers (safe to share)
- ✅ Effort index / competency weights (educational metadata, safe to share)
- ⚠️ Footer says "Confidential" (optional; teachers can redact if needed)

It does **not** contain:
- ✅ Student names or data
- ✅ Assessment results
- ✅ Personal information

**Safe to share** with colleagues, parents, students, or administrators.

## Tips & Best Practices

1. **Before first use:** Make sure your browser allows downloads (no blocker, permission granted)
2. **Filename:** Keep the auto-generated filename; it includes subject and grade for easy identification
3. **Multiple allocations:** You can allocate more chapters later; click "Allocate more chapters →" to add them
4. **Review often:** Export a fresh report each term/semester to track changes
5. **Backup:** Save a PDF copy of your final allocation for record-keeping

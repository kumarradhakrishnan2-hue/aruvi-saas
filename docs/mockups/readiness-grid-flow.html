Interactive prototype. Tap a cell to mark a class (fills the shortest duration). Hold a cell — touch long-press, or press-and-hold / right-click on desktop — to cycle it up to the next declared duration; tap a filled cell again to clear. Grids start empty. One grid per subject·grade; sections are fixed from setup. The flow loops grade by grade, then estimates annual teaching time the way the teacher already thinks about it.
